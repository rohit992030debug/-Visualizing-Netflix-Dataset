📊 Netflix Content Insights Dashboard

An interactive and visually engaging Netflix Content Insights Dashboard built to analyze trends, ratings, and content distribution across movies and shows.

This project provides deep insights into:

🎬 Top 10 titles based on IMDb votes and ratings
⏱️ Runtime vs IMDb score analysis
📈 Content distribution (Movies vs Shows)
🧾 IMDb voting patterns by age certification
📅 Filtering by release year and content type

🚀 Key Features
Dynamic filtering (Year, Type: Movie/Show)
Clean dark-themed UI for better visualization
Comparative analysis of ratings, votes, and runtime
Data storytelling through charts and KPIs

🛠️ Tech Stack
Power BI / Tableau (whichever you used)
Data Cleaning & Transformation
Data Visualization Techniques

📌 Use Case
This dashboard helps:
Understand audience preferences
Identify high-performing content
Analyze rating trends across different categories

💡 Insights Generated
Movies dominate the platform content share
Certain age certifications attract higher engagement
Strong correlation between votes and popularity
